Membership Categories

The PSF recognizes several categories of members:

  • Fellows are elected on merit
  • Emeritus Fellows are no longer active in the PSF
  • Contributing / managing members support the PSF's mission by actively engaging in bringing Python forward
  • Supporting members support the PSF by paying a yearly membership fee
  • Basic members show their support for Python by signing up to the PSF

See the membership FAQ for more information.

Only the first two membership categories are listed on this page, until we find a way to add the others as well.
